## 2024-05-22 - [Performance: Median Calculation]
**Learning:** `select_nth_unstable` is O(N) while `sort_unstable` is O(N log N). For finding median (or any k-th element), prefer `select_nth_unstable`. Also, if iterating, calculate min/max in the loop instead of scanning or sorting later.
**Action:** Check for full sorts when only median or top-k elements are needed.

## 2024-05-22 - [Build Script Side Effects]
**Learning:** The `xiv-gen` build script modifies `extra.toml` in the source tree based on environment/features. This causes dirty git state after running `cargo check`.
**Action:** Be careful not to commit `extra.toml` changes generated by local build environments.
